# AI Interview Coach

AI-powered web application that generates interview questions based on candidate resume and job role.

## Features
- Upload Resume (PDF)
- Enter Job Role
- AI Generated Interview Questions
- Streamlit Web Interface

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- NLP
- PDF Processing

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
